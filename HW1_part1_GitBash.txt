Linux terminal (GitBash) commands
1) Посмотреть где я - pwd
2) Создать папку - mkdir qa1
3) Зайти в папку - cd qa1
4) Создать 3 папки - mkdir qa11 qa12 qa13
5) Зайти в любоую папку - cd qa11
6) Создать 5 файлов (3 txt, 2 json) - touch f1.txt f2.txt f3.txt f4.json f5.json
7) Создать 3 папки - mkdir qa111 qa112 qa113
8. Вывести список содержимого папки - ls -la
9) + Открыть любой txt файл - vim f1.txt 
10) + написать туда что-нибудь, любой текст - нажать I и ввести текст Hello!!!
11) + сохранить и выйти - Esc + Shift ZZ
12) Выйти из папки на уровень выше - cd ../
13) переместить любые 2 файла, которые вы создали, в любую другую папку -   mv qa11/f1.txt qa11/f2.txt qa12/
14) скопировать любые 2 файла, которые вы создали, в любую другую папку -  cp qa11/f3.txt qa11/f4.json qa12/
15) Найти файл по имени - find . -name "g*"
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает -  cat g2.txt | grep Ann (вывели строчку, где есть слово 'Ann')
17) вывести несколько первых строк из текстового файла -  head g2.txt
18) вывести несколько последних строк из текстового файла - tail g2.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает -  less g1.txt (открывается отдельное окно с содержимым файла)
20) вывести дату и время - date (также можно настраивать формат и дату с помощью этой команды)
